    Tables

    Tables

    Tables are coordinates in a tabular format (essentially a matrix), optionally with named columns. They have various constructors, for direct construction and also for conversion from other types.

    Let

    x = linspace(0, 2*pi, 100)
y = sin.(x)

    You can pass these coordinates in unnamed columns:

    Plot(Table([x, y]); incremental = false)

    [.pdf], [generated .tex]

    or named columns:

    Plot(Table([:x => x, :y => y]); incremental = false)

    [.pdf], [generated .tex]

    or rename using options:

    @pgf Plot(
    {
        x = "a",
        y = "b",
    },
    Table([:a => x, :b => y]);
    incremental = false)

    [.pdf], [generated .tex]

    In the example below, we use a matrix of values with edge vectors, and omit the points outside the unit circle:

    x = linspace(-1, 1, 20)
z = @. 1 - √(abs2(x) + abs2(x'))
z[z .≤ 0] .= -Inf
@pgf Axis(
    {
        colorbar,
        "colormap/jet",
        "unbounded coords" = "jump"
    },
    Plot3(
        {
            surf,
            shader = "flat",
        },
        Table(x, x, z);
        incremental = false
    )
)
